# Quick Start Guide

## 10-Step Bootstrap

1. **Init:** Ensure `.gitignore`, `.gitattributes` exist
2. **Specs:** Fill `product.md` with vision
3. **Stack:** Define tools in `tech.md`
4. **Structure:** Map architecture in `structure.md`
5. **Profiles:** Create `profiles/dev.yaml` (gitignored)
6. **Artifacts:** Generate diagrams before code
7. **Tests:** Write tests alongside implementation
8. **Sync:** Update `status.md` after changes
9. **Review:** Follow PR discipline for `AGENTS.md`
10. **Deploy:** Check `hackathon_mode.json` for deadlines
