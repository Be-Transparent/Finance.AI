# Product Vision: Finance.AI (Diia.Фінанси)

## Goal

Finance.AI — державний AI-фінансовий сервіс для громадян України та ФОП, що допомагає оптимізувати особисті фінанси, отримувати податкові знижки, субсидії, пільги, гранти, кешбеки та можливості для інвестицій.

## Target Audience

- **Громадяни України:** Отримання податкових знижок, субсидій, пільг
- **ФОП (Фізичні особи-підприємці):** Оптимізація податків, гранти, автоматизація обліку
- **Держслужбовці:** Інтеграція з екосистемою Diia для надання фінпослуг

## Key Features

- **AI-класифікація транзакцій** — автоматичний розподіл витрат за категоріями
- **Парсинг чеків** — розпізнавання QR/PDF чеків для податкового обліку
- **Eligibility Engine** — перевірка права на пільги, субсидії, гранти
- **Recommendation Engine** — персоналізовані поради з економії, кешбеку, інвестицій
- **Інтеграція з Diia.Signature** — підпис документів через iQ
- **Integration Points:** Yana.Diia (AI-асистент), iQ (auth/eID), банки, ДПС
